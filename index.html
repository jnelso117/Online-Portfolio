My Work

Angry Dragons

This application was built using the Gamesalad game engine. GameSalad can be used to create applications for Android, iOS, or the Web. GameSalad does most coding for it's developers, so limited code was needed to develop this game. However, programming logic is needed as GameSalad uses boolean operators for many methods the creator calls. Our objective was to try and gamify the lessons learned in a physics lab.

Using GameSalad, 2014 Intern, Woodfin Glass and myself created the game for a client and will be used in a grant application here at UAB.

Developers: Woodfin Glass and Jeremy Nelson
